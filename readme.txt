Connect independent WooCommerce products as visual linked swatches for Elementor product templates.

== Description ==

VRED Linked Swatches lets you connect independent WooCommerce products and display them as visual swatches inside Elementor product templates.

It is useful when each color, finish, texture, material, or option has its own product page, images, price, stock, URL and SEO, but still needs to behave like a selectable product option.

Main features:

* Link related WooCommerce products from the product edit screen.
* Add optional display names, colors and texture images.
* Display linked products as visual swatches.
* Use color, split-color, texture image or product image fallback.
* Add a configurable Elementor swatches widget.
* Add a configurable Elementor trigger widget.
* Optionally open a VRED Elements panel from the trigger widget when VRED Elements is active.
* Use local frontend assets and a custom update system.

== Installation ==

1. Upload the plugin ZIP from Plugins > Add New > Upload Plugin.
2. Make sure WooCommerce and Elementor are active.
3. Activate VRED Linked Swatches.
4. Edit a product and open the Linked Swatches product data tab.
5. Select linked products and configure their optional swatch data.
6. Add the Linked Swatches or Linked Swatch Trigger widget to an Elementor product template.
